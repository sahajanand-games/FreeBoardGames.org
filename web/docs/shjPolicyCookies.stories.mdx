The main purpose behind the use of Cookies is to improve user experience and perform updates of the site. This project is [Open Source](https://github.com/sahajanand-games/SahajanandGames) and all the data that is processed is within the scope of this code. 

#### Google Analytics
The program uses a [google analytics plugin](https://github.com/react-ga/react-ga) to monitor usage. This helps us to monitor site usage and plan infrastructure requirements for the site. 

If you have any concerns regarding cookie use you can contact us via our email or directly on discord. We will try to answer your concerns and/or delete data upon request. Thank you for your understanding.
